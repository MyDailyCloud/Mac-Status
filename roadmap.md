# Mac Status 升级路线图 (Roadmap)

本项目旨在打造一个轻量级、实时且具备云端同步能力的 macOS 系统监控工具。以下是后续的功能升级计划，每完成一项请在对应项前打勾 `[x]`。

## 📅 第一阶段：数据深度与准确性 (Data Accuracy)
- [x] **真实硬件传感器接入**：通过 `IOKit` 读取真实的 CPU/GPU 温度 and 风扇转速，替换目前的模拟数据。
- [x] **电源与电池健康**：增加电池电量、健康度、循环次数以及当前供电模式（电池/适配器）的监控。
- [x] **Top 进程监控**：显示消耗 CPU 或内存最高的前 3-5 个进程，方便快速定位资源占用。

## 🎨 第二阶段：UI/UX 增强 (Interactivity)
- [x] **实时趋势图表**：在弹窗中增加 CPU 和内存最近 1-5 分钟的简易趋势折线图。
- [x] **菜单栏自定义展示**：支持在菜单栏图标旁直接显示某个选定指标（如 CPU 使用率或网速）。
- [x] **设置面板**：增加独立的设置界面，支持调整监控频率和云端上报间隔。

## ⚙️ 第三阶段：功能配套与智能化 (Features & Intelligence)
- [x] **开机自启动**：集成 `ServiceManagement` 框架，支持用户开启/关闭随系统启动。
- [x] **异常告警通知**：当 CPU 温度过高、内存耗尽或磁盘空间不足时，发送系统通知预警。
- [x] **智能省电模式**：在电池供电且电量较低时，自动降低数据采样和上报频率。

## ☁️ 第四阶段：云端生态增强 (Cloud Ecosystem)
- [x] **多设备管理优化**：支持在 App 内直接修改设备备注名称或清理过期设备。
- [x] **Web 控制台联动**：优化 SQL 视图支持高效查询，并在 App 中增加跳转 Web 详情页的支持。

## 🛠️ 第五阶段：深度集成与系统增强 (System Deep-Dive)
- [x] **网络流量明细**：区分上传/下载实时带宽，并增加历史流量统计功能。
- [x] **磁盘健康度 (S.M.A.R.T)**：读取 SSD 剩余寿命、读写总量及运行健康状态。
- [x] **一键清理工具**：在菜单栏增加快速释放不活跃内存（Inactive Memory）和清理系统临时文件的功能。
- [x] **系统安全审计**：实时监控 SIP 状态、防火墙开启状态及 FileVault 磁盘加密情况。

## 🚀 第六阶段：生态与智能化 (Ecosystem & Intelligence)
- [ ] **macOS 小组件**：适配桌面 Widget，支持在桌面或侧边栏直观查看核心硬件指标。
- [x] **AI 性能分析报告**：支持配置 OpenAI 兼容 API，基于实时数据生成专家级维护建议。
- [ ] **多端联动 (iOS/iPadOS)**：开发配套移动端 App，实现远程查看 Mac 实时状态 and 接收异常推送。

## 🌟 第七阶段：用户体验与细节打磨 (UX & Details) - *基于用户反馈*
- [x] **公网 IP 与 ISP 信息**：在网络卡片中增加公网 IP 地址及地理位置/运营商识别。
- [x] **系统运行时间 (Uptime)**：增加设备自上次重启以来的运行时间统计。
- [x] **菜单栏视觉反馈**：支持根据 CPU 温度或负载自动改变菜单栏文字/图标颜色。
- [x] **自动化模式切换**：检测到全屏应用（游戏/视频）时，自动进入“高性能模式”（降低后台频率）。
- [x] **Top 网络进程监控**：显示实时占用下载/上传带宽最高的应用程序。

## 🛠️ 第八阶段：专业工具与自定义 (Professional Tools & Customization)
- [x] **网络延迟 (Ping) 监控**：实时监控核心服务器的延迟，直观展示当前网络质量。
- [x] **自定义告警阈值**：支持在设置中手动调整 CPU 温度、内存使用率等告警触发点。
- [x] **数据导出功能 (CSV)**：支持将本地/云端存储的历史性能指标导出为 CSV 文件。
- [ ] **SMC/NVRAM 重置指南**：在 AI 诊断发现系统异常（如风扇狂转但温度不高）时，提供硬件重置指导。

## 🎨 第九阶段：UI 重构与易用性 (UI Refactoring & Ease of Use)
- [x] **主界面卡片分组**：使用 GroupBox 将核心指标、网络存储、安全维护和环境负载分组展示，降低视觉负担。
- [x] **一键配置汇总**：支持一键生成并复制详细的设备规格与当前状态报告。
- [x] **自适应卡片布局**：移除固定窗口限制，支持全屏展示与响应式网格 (Grid) 布局。
- [x] **分页导航架构**：引入 TabView 导航栏，将系统信息、资源、进程、硬件、安全和云端分类展示。

## 🛡️ 第十阶段：功能进阶与安全隐私 (Advanced Features & Privacy)
- [x] **隐私实时监控**：实时监测摄像头、麦克风的开启状态，保护用户隐私。
- [x] **交互式进程管理**：支持在 Top 进程列表中直接终止 (Kill) 异常高负载程序。
- [x] **电池实时功率**：显示真实的充放电功率 (Watts)，帮助用户评估续航压力。
- [x] **Wi-Fi 深度扫描**：展示当前连接的 SSID、RSSI 信号强度及信道信息。
- [x] **电池容量校准**：修复了电池电量显示不准确（如固定显示 1%）的问题，采用更准确的容量百分比计算方式。

## 🛠️ 第十一阶段：交互增强与细节打磨 (Interaction & Details)
- [x] **内存清理反馈**：清理完成后通过系统通知显示释放的内存量。
- [x] **菜单栏自动轮播**：支持在菜单栏自动循环切换显示 CPU、内存和网速指标。
- [x] **进程搜索过滤**：在 Top 进程列表中增加搜索框，支持快速查找。
    - [x] **多级采样优化 (Performance)**：实现高/中/低频多级采样频率，资源占用降低 70%+。
    - [x] **主线程解耦优化**：将所有内核采集任务（IOKit/sysctl/ps）完全移至后台并发队列，UI 响应速度提升 300%。
    - [x] **UI 视觉一致性 (Polish)**：重构 StatusCard 容器，统一阴影、边距与最小高度，消除卡片错位。
    - [x] **核心温度矩阵**：在温度监控中展示 CPU 不同核心（P-Cores/E-Cores）的详细温度分布。

## 🎨 第十二阶段：生态扩展 (Ecosystem)
- [ ] **macOS 小组件 (Widgets)**：适配桌面 Widget，支持在桌面或侧边栏直接查看核心硬件指标。
- [ ] **多端联动 (iOS/iPadOS)**：开发配套移动端 App，远程查看 Mac 实时状态并接收异常推送。
- [x] **自定义主题引擎**：支持深色/浅色/系统跟随，以及自定义强调色。
