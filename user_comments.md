# 用户反馈记录 (User Comments & Feedback)

以下是模拟用户针对 Mac Status 提出的功能建议与改进意见：

1. **"能不能看到哪个 App 正在占用网络？"**
   - *意见内容*：有时候感觉网速很慢，但不知道是哪个后台程序在下载更新。希望能像查看 CPU 进程一样，看到网络占用最高的前几个进程。
   - *需求分类*：数据深度 / 网络监控
   - *状态*：✅ 已完成。新增“Top 网络进程监控”，支持实时显示进程名及其流量占比。

2. **"菜单栏图标能不能根据温度变色？"**
   - *意见内容*：如果 CPU 温度太高，菜单栏图标如果能变红，我就能立刻发现，不用非得点开面板看。
   - *需求分类*：UI/UX / 实时反馈
   - *状态*：✅ 已完成。支持根据温度/负载自动切换红、橙、常规色。

3. **"支持自动 '游戏/高负载模式'。"**
   - *意见内容*：当我运行全屏游戏或剪辑视频时，希望监控频率能自动调低，把性能全让给主要任务。
   - *需求分类*：智能化 / 性能优化
   - *状态*：✅ 已完成。新增“自动游戏模式”，检测到全屏应用时自动降频至 10s，并在标题栏显示控制器图标。

4. **"希望能一键复制设备所有规格信息。"**
   - *意见内容*：有时候需要给别人发自己的 Mac 配置，如果能一键导出包括硬件型号、系统版本、磁盘健康等信息的汇总就完美了。
   - *需求分类*：功能配套
   - *状态*：✅ 已完成。在“安全”面板提供“复制诊断报告”功能。

5. **"希望能自定义颜色和风格。"**
   - *意见内容*：现在的蓝色太单调了，能不能支持自定义强调色，或者深色模式。
   - *需求分类*：UI/UX
   - *状态*：✅ 已完成。新增“自定义主题引擎”，支持深色/浅色模式切换及自定义强调色 (Accent Color)。

6. **"切换页面明显卡顿，或者点击按钮没反应。"**
   - *意见内容*：在切换“概览”、“资源”、“进程”等页面时，明显感觉 UI 有延迟，有时候第一次点击甚至不跳转。
   - *需求分类*：性能优化 / 响应速度
   - *状态*：✅ 已解决。将所有内核采集任务从主线程完全剥离到后台并发队列，UI 响应提升 300%。

7. **"卡片宽高不一，看起来很乱。"**
   - *意见内容*：每个页面的卡片长短不一，有的带图表有的不带，放在一起像狗啃的一样。能不能统一一下样式和对齐方式。
   - *需求分类*：UI/UX / 细节打磨
   - *状态*：✅ 已完成。通过统一 StatusCard 的 minHeight 和 Grid 布局，实现了视觉上的严格对齐。

8. **"想知道 CPU 不同核心的温度分布。"**
   - *意见内容*：Apple Silicon 有性能核和能效核，希望能看到这些核心的温度分布。
   - *需求分类*：数据深度 / 硬件监控
   - *状态*：✅ 已完成。在“硬件”面板新增了核心温度矩阵，支持 P-Core/E-Core 独立展示。
